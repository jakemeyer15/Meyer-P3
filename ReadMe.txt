   Pixelator
   Jake Meyer

   This is a program that converts any image to a pixelated image or 8-bit image.

   Steps to run program:
1. Navigate to the Meyer_proj3 directory
2. Type into the command prompt: "make compile" (all files will be compiled)
3. Type into the command prompt: "make run" (new window will appear with pixelator program)

   For further instructions, consult "Pixelator_UserManual.pdf" or the File > Get Started
   option.
